Function Descriptions:

    printOperationType: This function prints the symbol of the selected operation type (+, -, x, /).
    selectOperationType: Allows the user to select an operation type (addition, subtraction, multiplication, division, or random).
    selectDifficultyLevel: Allows the user to select a difficulty level (easy, medium, hard, or random).
    generateRandomNumber: Generates a random number between a given range.
    setTerminalColor: Changes the terminal color based on whether the answer is correct or incorrect.
    getNumberOfQuestions: Prompts the user to input the number of questions for the quiz.
    convertOperationTypeToString: Converts the operation type to its corresponding string representation for display.
    convertDifficultyLevelToString: Converts the difficulty level to its corresponding string representation for display.
    calculator: Performs the selected mathematical operation on two numbers.
    generateRandomDifficultyLevel: Generates a random difficulty level.
    createQuestion: Creates a question based on the selected difficulty level and operation type.
    createExam: Generates multiple questions and stores them in the exam.
    getUserAnswer: Accepts the user's answer to a question.
    displayQuestion: Displays the question to the user.
    compareAnswer: Compares the user's answer to the correct answer and updates the result.
    askQuestionsAndCheck: Loops through all questions, displays them, gets the answers, and checks them.
    finalStatus: Returns the result of the game, whether the user passed or failed.
    printResultTable: Prints the result table, showing the number of correct and incorrect answers, difficulty level, and operation type.
    runGame: Manages the flow of the game (getting the number of questions, difficulty level, operation type, creating the exam, asking questions, and displaying results).
    resetTerminal: Resets the terminal's appearance.
    start: Starts the game, allowing the user to play again if desired.
